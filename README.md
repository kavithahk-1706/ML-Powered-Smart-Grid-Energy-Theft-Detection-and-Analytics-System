# ML Powered Smart Grid Energy Theft Detection and Analytics System

## Data Directory

### Dataset Files (not in git)

Due to file size, CSV files are not tracked in git.

**Raw data:** `data/raw/dataset1.csv` (560,655 rows)
**Processed data:** `data/processed/dataset_final.csv`

### How to get the data:

1. Download from [Google Drive link](https://drive.google.com/file/d/1e6tSdAgFS-y_hLO0wwyoTvO8eUdH0Dwt/view?usp=sharing) 
2. Place in the appropriate directory
3. Run preprocessing notebook to generate processed data

### Directory structure:
```
data/
├── raw/
│   └── dataset1.csv (not in git)
├── processed/
│   └── dataset_final.csv (not in git)
└── README.md (this file)
```
